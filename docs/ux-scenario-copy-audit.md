## 3. 評価時に見るべき観点

- やりたくなるか
- 何をすればいいか分かるか
- 文言が自然か
- 決めつけ感がないか
- 不安を煽っていないか
- 押した後に迷わないか
- また開きたくなるか
- 猫ごとの文脈が分かりやすいか
- 記録が義務に感じないか
- アプリとして完成度があるか
