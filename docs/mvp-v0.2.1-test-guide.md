# にゃるほど MVP v0.2.1 テストガイド

## テスト目的

猫の正解を当てることではなく、飼い主の迷いが減る体験になっているかを確認する。

## テスター向け説明文

猫の行動から、いまの気持ちを少し見立てて、次にできそうなことを提案するアプリです。

まずは「うちの猫、どんなタイプ？」から始めて、気軽に触ってください。

正解を当てるものではないので、「違うかも」と思ったらその感覚も大事なフィードバックです。

## 注意事項

- 医療判断をするアプリではありません
- 体調が気になる場合は必要に応じて専門家に相談してください
- 入力した内容は検証用に使う可能性があります
- 現時点では端末ごとの保存です
- 別端末ではデータが引き継がれません

## テストしてほしい流れ

1. 初回診断オンボーディングを触る
2. 3問後の結果を見る
3. 可能なら15問まで答える
4. ホームで今の様子を1つ残す
5. ちょっと気になる？から1つ選ぶ
6. 診断結果でCTAまたは違うかもを押す
7. ねこページを見る

## フィードバック項目

- 最初にやってみたいと思ったか
- 3問後の結果は早すぎる/決めつけに感じたか
- もう少し答えてみたいと思ったか
- ホームで何をすればいいか分かったか
- 記録が面倒/義務に感じたか
- 診断結果で次に何をすればいいか分かったか
- 文言が不安を煽っていなかったか
- 違うかもを押しやすかったか
- また開きたいと思ったか
- 分かりにくかった箇所
- 良かった箇所

## 判定基準

### 高優先度

- 初回で離脱する
- 何をすればいいか分からない
- 不安を煽る
- 決めつけに見える
- 記録が義務に感じる
- 猫ごとの文脈が混ざる

### 中優先度

- 文言が少し硬い
- もっと楽しくできそう
- ボタンの意味が少し分かりづらい

### 低優先度

- 見た目の好み
- 将来機能の要望
