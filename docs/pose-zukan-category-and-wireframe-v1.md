# ポーズ図鑑カテゴリ v1 と いっしょタブUIワイヤー方針

## 1. 結論

MVP初期のポーズ図鑑カテゴリは12個にする。

選定基準:

- 猫あるあるとして伝わりやすい
- 写真を撮りたくなる
- 見つけた時に嬉しい
- SNS/LPで見せやすい
- 医療/異常っぽく見えない
- コンプリート圧が強すぎない
- 手動追加でも成立する
- 将来AI分類しやすい

ポーズ図鑑は、にゃるほどを図鑑アプリへピボットするためではなく、暮らしログと診断体験に「見つける」「たまる」「ふりかえる」動機を重ねるための要素として扱う。

## 4. カテゴリのグルーピング

12カテゴリは、いっしょタブで見やすいように以下の4グループに分ける。

### くつろぎ

- へそ天
- 香箱座り
- まるまり
- おててないない

### ねむい

- ごめん寝
- 顔だけ出す

### あそび・ふれあい

- のびー
- ふみふみ
- すりすり

### おきにいりの場所

- 箱入り
- 液体化
- 窓辺監視

分類名は硬くしすぎない。図鑑は「分類する場所」ではなく「見つけた姿を眺める場所」として扱う。

## 5. 「みつけた」表現ルール

図鑑では以下の表現を使う。

使う:

- みつけた
- 最近みつけた
- 初めてみつけた
- この子らしい
- また見られた

避ける:

- 達成
- コンプリート
- 未達成
- ミッション
- ノルマ
- レベル
- ランキング

理由:
記録を義務にせず、発見として感じてもらうため。

## 6. いっしょタブの静的UIワイヤー方針

いっしょタブは以下の順番にする。

### 6-1. 最近みつけた

最上部。写真/仮サムネ中心。

表示例:

最近みつけた
へそ天
箱入り
のびー

初期データがない場合:

まだこれから
かわいい瞬間を見つけたら、ここに増えていきます。

ただし、空状態が寂しすぎないようにする。空の枠だけを並べず、やさしい案内にする。

### 6-2. ポーズ図鑑

主役。

表示例:

ポーズ図鑑
3つみつけた

12カテゴリをグリッドで表示。見つけたものは明るく、未発見は薄くする。ただし、未発見を「未達成」と見せない。

表示ルール:

- 見つけた: 写真 or アイコン + 表示名
- まだ: 薄いアイコン + 表示名
- カウントは「3/12」ではなく「3つみつけた」寄り

### 6-3. この子らしさ

AI的な軽いまとめ。

表示例:

むぎは夜にへそ天しがち
最近は箱入りが増えています
窓辺監視がよく見つかっています

注意:

- 断定しない
- かわいく短く
- 医療/異常っぽくしない

### 6-4. 思い出タイムライン

時系列。

表示例:

5/10 初めてのへそ天
5/22 箱入りブーム
6/03 窓辺監視多め

初期は仮データでOK。将来、写真・手動追加・AI候補と接続する。

## 7. いっしょタブで弱める/削るもの

主役にしないもの:

- 関係スコア
- バッジ
- 見た回数
- 0日表示
- 達成率
- 連続記録
- ランキング

理由:
猫との暮らしより、アプリのタスク感が出やすいため。

## 8. ホームとの接続方針

ホームには図鑑を大きく出さない。

ホーム側に出すなら小さく:

最近みつけた
へそ天・箱入り・のびー

または、猫カードの下に1行だけ。

ただし、ホームは以下の役割を維持する。

- 今日の猫をのぞく
- 見たままを残す
- 気になる時に診断する

図鑑詳細はいっしょタブに置く。

## 9. 最初のUI実装スコープ候補

次にUI実装するなら、まず静的UIでよい。

Phase T3候補:

- いっしょタブに静的な図鑑UIを作る
- localStorage保存なし
- 写真アップロードなし
- AI分類なし
- 仮データで「3つみつけた」表示
- 12カテゴリのグリッドを表示
- 最近みつけた / この子らしさ / タイムラインも仮表示

目的:
見た目と情報設計の検証。

## 10. 将来の保存方針

今すぐDBは作らない。

将来必要になるデータ:

- pose_id
- cat_id / local_cat_id
- image_url
- detected_by_ai
- confirmed_by_user
- discovered_at
- memo
- source_event_id

ただし、現時点では実装しない。

localStorage MVPでは、将来的に以下のような形を検討する。

```json
{
  "pose_discoveries": [
    {
      "localCatId": "local-cat-xxx",
      "poseSlug": "belly_up",
      "discoveredAt": "ISO",
      "imageUrl": null,
      "source": "manual"
    }
  ]
}
```
