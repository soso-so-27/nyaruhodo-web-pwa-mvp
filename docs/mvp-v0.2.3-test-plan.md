# MVP v0.2.3 実利用テスト計画

## 1. テスト目的

MVP v0.2.3 は、機能が動くかだけでなく、日常の中で自然に使えるかを確認する段階とする。

確認したいこと:

- 毎日自然に開けるか
- 今日タブで迷わず残せるか
- 記録が義務に見えないか
- 初回の「みつけた」表示がうれしいか
- コレクションを見る理由があるか
- ねこタブが管理場所として分かるか
- 猫ごとの文脈が混ざらないか
- 不安を煽る表現がないか

にゃるほどは、猫の正解を当てるアプリではなく、猫を見ていたら暮らしがたまっていくアプリとして検証する。

## 2. テスト期間

- まずは3日間
- できれば7日間
- 1日1〜3回程度
- 無理に入力しない
- 開きたい時、見えた時、気になった時だけ使う

## 3. テスト端末

- iPhone実機
- PWAホーム画面追加状態
- Safari / Chrome どちらでもよい
- 主戦場はスマホ実機

確認時は、ブラウザのアドレスバー込みではなく、PWAとして開いた時の下部ナビや余白も見る。

## 4. /home テスト観点

確認項目:

- 最初に開いた時、何をすればよいか分かるか
- 猫カードが重すぎないか
- `さいきん / いま / 1日ミニマップ` が見ていて楽しいか
- `いつもの様子` を押すのに迷わないか
- `ちょっと気になる` が不安を煽らないか
- concern を押しても `/diagnose` に飛ばないのが自然か
- `✓ 残しました` が分かりやすいか
- `コレクションで「◯◯」をみつけました` がうるさくないか
- 保存失敗時の表示が分かりやすいか
- 今日 / コレクション / ねこ の下部ナビが窮屈でないか

特に見ること:

- 押した後に「保存できた」感があるか
- 入力が作業ではなく、見たままを残す行為に見えるか
- 気になる系を押しても、診断に連れて行かれないことが自然か

## 5. /collection テスト観点

確認項目:

- コレクションが設定画面に見えないか
- 「たまったものを見る場所」に見えるか
- 4列グリッドが読みやすいか
- `よくわからない` が折り返して気にならないか
- `みつけた` バッジが強すぎないか
- `まだこれから` が失敗やロックに見えないか
- 最近みつけた と このごろ が重複して見えないか
- このごろ が分析・診断っぽく見えないか
- concern 系が続いても不安を煽らないか
- 空状態が寂しすぎないか

特に見ること:

- 「最近みつけた」が直近で増えたものに見えるか
- 「ポーズコレクション」が達成表ではなく、暮らしの棚に見えるか
- 「このごろ」がスコアではなく、軽いひとことメモに見えるか

## 6. /cats テスト観点

確認項目:

- ねこタブが管理場所として自然か
- 猫追加が分かりやすいか
- 名前変更が分かりやすいか
- 毛色選択が自然か
- 毛色別アバターで「うちの子感」が少し出るか
- コレクションや分析要素が混ざって見えないか

特に見ること:

- /cats はプロフィール・管理の場所として軽く見えるか
- 今日やコレクションの役割を奪っていないか
- 猫追加や名前変更が毎日使う操作のように目立ちすぎていないか

## 7. 複数猫テスト観点

確認項目:

- 猫を切り替えても記録が混ざらないか
- active cat ごとにコレクションが分かれるか
- 別猫で同じ pose を初回記録した時に「みつけました」が出るか
- 猫ごとのアバター差が分かるか
- 切り替え導線が重くないか

確認例:

- ミケで `ねてる` を残す
- むぎに切り替える
- むぎのコレクションにミケの `ねむる` が混ざらないことを見る
- むぎで初めて `ねてる` を残した時、むぎ側で初回の「みつけました」が出るかを見る

## 8. 観察メモ形式

実利用中は、以下の形式で短く残す。

```text
日付:
猫:
開いたタイミング:
押した項目:
その時の気持ち:
良かった点:
迷った点:
重く感じた点:
また開きたいと思ったか:
気になった文言:
スクショ要否:
```

メモは完璧に埋めなくてよい。違和感があった時だけでも十分。

## 9. 判断基準

### 続ける

- 今日タブを自然に開けた
- 1日1回以上、無理なく残せた
- コレクションを見に行く理由が少しあった
- 不安を煽らなかった
- 猫ごとの違いが少し感じられた

### 直す

- ホームが重い
- 何を押すか迷う
- コレクションが棚ではなく分析画面に見える
- 4列グリッドが読みにくい
- 初回みつけた表示がうるさい
- concern 系が不安に見える
- ねこタブに行く理由が分からない

### まだ進めない

- 手動みつけた追加
- 写真機能
- 通知
- 課金
- 家族共有
- DB本格化

これらは、実利用で「自然に続く」感覚が見えてから判断する。

## 10. テスト後の次アクション候補

優先順:

1. `/collection` の4列グリッド微調整
2. `よくわからない` の表示調整
3. 最近みつけた / このごろ の重複調整
4. ホーム猫カードの重さ調整
5. 初回みつけた表示の文言調整
6. 実利用後に、通知・写真・家族共有の優先度を再判断

慎重に扱うもの:

- 手動みつけた追加
  - 二重入力感が出る可能性がある
  - 今はまだ入れず、実利用テスト後に判断する
