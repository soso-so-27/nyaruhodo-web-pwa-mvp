# MARKETING-CANON v0.1

対象: ねてるねこ Instagram / 外向き告知文面。
初版: 2026-07-04

このCANONは、ねてるねこのInstagram投稿と公開前後の告知文面を、静かで個人的な声に保つための基準です。アプリ本体の仕様ではなく、外に出す言葉の判断基準として使います。

## 1. 目的

中西が、むぎの写真や開発の断片を外に出すとき、広告・煽り・成長施策の文体に流れないようにする。

投稿は「宣伝」ではなく、「いま作っているものと、その理由を少しだけ分ける」ためのものです。最終判断は常に中西が行います。

## 2. 種別

### むぎ日常

むぎの寝顔、近況、写真そのものの情景を主役にする投稿。

- 写真の情景から入る
- 近況は短く添える
- アプリの話を入れる場合も、写真の余韻を壊さない

### build in public

ねてるねこの開発過程、迷い、決めたことを静かに共有する投稿。

- 作っている理由を個人の言葉で書く
- 機能紹介より、考えたことや迷ったことを置く
- 成果の誇示ではなく、途中経過の打ち明けにする

## 3. 文体・禁止表現

### 文体

- です・ます調の打ち明けにする
- 主語は「私」または無主語にする
- 企業文体、宣言体、キャンペーン文体にしない
- 写真の情景 -> メモの断片 -> 必要なら静かな締め、の順にする
- オチ、教訓、強い結論をつけない
- 短文基調にする
- 体言止めを多用しない

### 禁止語リスト

完全一致と自然な活用形を検出対象にする。

| 分類 | 禁止語 |
| --- | --- |
| 死関連 | 一生, 最期, 見送る, 見送り, 虹の橋, 天国 |
| 煽り | 今だけ, お早めに, 残りわずか, 見逃すな, 必見, バズ, 緊急 |
| 誇示 | 突破, ランキング, 話題の, 大人気, 急上昇 |
| 依頼 | 拡散, シェアお願い, フォローお願い, いいねお願い |
| ゲーミフィケーション | 連続記録, ストリーク, ポイント, ランク, 達成 |
| リファラル報酬 | 紹介で, 招待特典 |

### 形式

- 絵文字は使わない
- ハッシュタグは3個以内にする
- 煽り系タグは禁止する
- 文字数は400字以内を目安にする

## 4. 語彙ロック

アプリ内動作に言及する場合だけ適用する。

| 概念 | 使う語 |
| --- | --- |
| 寝顔写真 | ねがお |
| 通知・配信 | ねこだより / とどく |
| 開封 | ひらく |

「毎日」は、必要があれば「一日一枚」と言い換える。ただし強制ではなく、指摘レベルとする。

## 5. 告知フェーズ

| フェーズ | 目的 | 出せるもの | 出せないもの |
| --- | --- | --- | --- |
| P1 | まだ静かに作っていることを共有する | 寝顔のためのアプリを作っていること、開発の迷い、写真の近況 | アプリ名「ねてるねこ」, リリース日, ストアリンク |
| P2 | 名前は出せるが、公開日は固定しない | アプリ名, 方向性, 事前の関心表明 | リリース日, ストアリンク |
| P3 | 公開・案内 | アプリ名, リリース日, ストアリンク | 数字誇示、煽り、禁止語 |

P3でも数字誇示は禁止する。ランキング、突破、大人気などの実績演出は使わない。

## 6. 投稿判断

CANONは最終判断を自動化しない。投稿するか、どの文にするか、どこまで言うかは中西が決める。

生成AIやCodexが文案を作る場合は、冒頭でドラフトであることを明示する。

## 7. 例

良い例:

> ひさしぶりの投稿です。むぎは元気で、あいかわらずよく寝ています。
> ストーリーズで気づいた方もいるかもしれませんが、いま、この寝顔のためにアプリをひとつ作っています。

悪い例:

> ついに完成間近！ むぎのアプリ、絶対見逃さないで！フォロワーのみんなに最速でお知らせします

違反: 煽り、依頼文体、強すぎる感嘆、広告文体。

悪い例:

> 【お知らせ】この度、新サービスをリリースする運びとなりました。

違反: 宣言体、企業文体、打ち明けの文体ではない。

## 8. 委任する（定型）

Codexに委任してよい定型作業:

1. 写真の説明と一言メモから、IGフィード投稿のキャプション案を2種作る
2. 作成した案をCANON §3・§4・§5に照らして検査し、検査結果を添える

Codexに委任しない作業:

- 投稿の可否判断
- 最終文言の決定
- 投稿タイミングの決定
- コメントやDMへの返信判断
