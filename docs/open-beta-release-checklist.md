# ねてるねこ 告知前チェックリスト

告知してよい状態かを確認するための最小チェック。
毎回、`main` 反映後の本番URLで確認する。

## 1. 配信前の技術チェック

- [ ] `git branch --show-current` が `codex/hitotoki-exchange`
- [ ] `npm run typecheck` が通る
- [ ] `npm run build` が通る
- [ ] `npm audit --omit=dev` が High 0 である
- [ ] `npm run check:release` が通る
- [ ] `supabase migration list` で Local / Remote が一致している
- [ ] Vercel の本番デプロイが成功している
- [ ] 本番URLで `/home` `/collection` `/cats` `/settings` が開ける

### ベータ公開前のセキュリティ/PWAメモ

- `npm audit --omit=dev` の Moderate 2 は `postcss` 経由。`npm audit fix` が Next 9 系への破壊的downgradeを提案するため、ベータでは未対応として記録する。
- client-visible signed URL は短命化する。`exchange` 経路は10分、`photo-storage/signed-url` 経路は1時間を上限にする。
- Service Worker は `/api/*`、Supabase signed URL、`token` 付きURL、authorization付きrequestをキャッシュしない。ユーザー写真/API JSON/認証済みレスポンスをCache Storageへ保存しない。
- `sleeping-delivery-pool-guards.spec.ts` の配信日検証は Supabase public env が必要。env未設定時はrelease gateでは明示skip、env設定済み環境では通す。

## 2. iPhone PWAの起動

- [ ] Safariで本番URLを開く
- [ ] ホーム画面に追加する
- [ ] ホーム画面のPWAから起動する
- [ ] Googleアカウントで接続する
- [ ] PWAを閉じて再起動してもログインが残る
- [ ] Safari/Web側ではなく、PWA側で操作していることが分かる

## 3. 保存の確認

- [ ] 寝顔を1枚入れる
- [ ] `とったねがお` の数が増える
- [ ] 必要なら「とどかないようにする」を切り替えられる
- [ ] 設定で「この端末のデータを保存」を実行する
- [ ] 保存結果に `とったねがお` の件数が出る
- [ ] エラーが出ていない

## 4. 復元の確認

- [ ] 別PWA、または新しく追加したPWAを開く
- [ ] 同じGoogleアカウントで接続する
- [ ] 「アカウントから復元」を実行する
- [ ] 猫が戻る
- [ ] `とったねがお` が戻る
- [ ] `とどいたねがお` が戻る
- [ ] アルバム写真が戻る
- [ ] 復元結果に写真の件数が出る
- [ ] エラーが出ていない

## 5. 削除と非表示の確認

- [ ] アルバム写真を削除する
- [ ] 復元をもう一度実行しても、その写真が戻らない
- [ ] `とったねがお` を削除する
- [ ] 復元をもう一度実行しても、その寝顔が戻らない
- [ ] `とどいたねがお` を非表示にする
- [ ] 復元をもう一度実行しても、その寝顔が戻らない
- [ ] 通報は受け取った時点の画面だけに出ている

## 6. 写真体験の確認

- [ ] 寝顔を入れる導線が最初に分かる
- [ ] 写真選択後の確認画面で縦写真が崩れない
- [ ] 写真選択後の確認画面で横写真が崩れない
- [ ] `自分だけ` と `とどく` の意味が誤解されない
- [ ] 受け取った寝顔を `とっておく` できる
- [ ] 受け取った寝顔を閉じられる

## 7. 設定と安心

- [ ] 保存と復元が何をするか分かる
- [ ] 失敗時に原因が少し分かる
- [ ] 写真の保存場所について読める説明がある
- [ ] プライバシーポリシーへ行ける
- [ ] 利用規約へ行ける
- [x] データ削除導線が見える
- [ ] 正式な問い合わせ先を告知文、プロフィール、設定画面のいずれかに入れる

## 8. 告知文

- [ ] 1文で体験が分かる
- [ ] 長い説明を読ませない
- [ ] 「うちの寝顔を1枚おくと、どこかの寝顔が1枚届きます」を基本にする
- [ ] ベータ版であることを添える
- [ ] 不具合報告先を添える（設定画面には差し替え用の表示枠あり）
