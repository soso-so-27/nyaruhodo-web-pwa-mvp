最終更新: 2026-06-16

この文書は、ホーム / まいにち / うちのこ / 命名 / コピーの一次資料。バックエンドの JST 検証、モデレーション、配信プールは既存実装に従い、本書は連携点のみ記す。

## 0. 目的

- 芯は「自分の猫をもっと見て、愛おしむ」。
- よその眠り猫の交換はきっかけであって目的ではない。
- フィード、フォロワー、ランキング、ストリーク、各種カウントは作らない。
- 1日1回、20:00 JST に1通だけ届く。
- 判断に迷ったら「自分の猫を見る回数が増えるか」「静けさを壊さないか」で決める。

## 2. ホーム

### 2.1 特大の額装

- 自分の猫の写真をほぼ画面いっぱいに出す。
- 写真の四辺に細い均一のふちを残す。
- 写真そのものは無加工。色温度の補正、ティント、暗転を写真にかけない。
- 時刻の光は写真の上ではなく、ふちが担う。

### 2.2 State2 の時刻連動の光【2026-07-07 実装準拠へ改訂】

- 時刻の光は、ふち限定ではなく**ホーム全体**で表現する（実装が正）。
- 実装: 時刻補間されるambient wash（`HOME_DAYLIGHT_ANCHORS`）＋時間帯別の紙背景画像
  （night/dawn/morning…、`getHomeBackgroundImage`）。`src/components/home/HomeDeskModel.tsx`
- 旧記述（ふちの色の連続変化・夜のグローをふちに限定）は未実装の旧案として無効。
  DESIGN-CANON §1.1「時刻はホーム全体のambient paper gradientで表現」と整合。

### 2.3 常時の状態

- ホームは常に最新の自分の猫を表示する。
- 今日撮っていなければ前日の猫を表示する。
- 通常ホームを空にしない。

### 2.4 1日の流れ

| 時間帯 | 表示 | ボタン/補助 |
| --- | --- | --- |
| あさ | きょうも すやすや | ねがおを とる |
| ひる（送った後） | むぎを おくった | よる、よその ねがお が とどく |
| ゆうがた | （17時の待機行は廃止）「おくった／よる8時に とどく」を20時まで維持 | E2Eガードあり |
| よる（到着） | とどいた | そっと ひらく |
| 会う（開封後） | どこかの ねこも、ねてた | むぎの ねがおも、どこかへ |
| 夜・終い | おやすみ | また、あした |

### 2.5 撮るエンジン

- 今日の自分の猫を送ると、夜によそのねこが届く。
- 送らない日は今夜のねこが来ないだけ。罰、ストリーク切れ、罪悪感として扱わない。
- 朝のそっとした合図を一つ添える。
- ホームを空にして急かす手法は採らない。

### 2.6 20:00 到着からまいにちへ

- 20:00 配信が届くと、ホームで開封の所作から「会う」。
- フルの開封セレモニーは1日1回の到着時のみ。
- 会ったあと、とどいた猫はまいにちへ収まり、ホームは自分の猫に戻る。
- 「むぎの ねがおも、どこかへ」は開封時の一度きりの静かな一拍。カウントにしない。

### 2.7 空状態

- 初回の一枚はオンボードで撮るため、通常ホームは空にならない。
- 例外時は紙と時刻の光の空の額に「はじめての ねがおを とろう」「とると、ここに むぎが すむ」。

### 2.8 クローム

- 日付は小さく、カメラは「ねがおを とる」。
- ホームには余計なナビを置かない。
- 夜はクロームを明色に、それ以外は暗色にする。

### 2.9 要確認

- ふちの太さ。
- 写真の基準アスペクト。
- 夜のグロー強度。
- ゴールデン/夜の色味の最終値。

## 3. まいにち

### 3.1 性質

- フラットなフィードではない。当月のフォトボード。
- 写真が有機的に貼られる。
- 空セルを作らない。
- おくった / とどいた のトグル。既定はおくった。
- ぜんぶ/all 面は作らない。

### 3.2 長期の堅牢性

- 各月は縦スクロールで1ヶ月分を表示する。
- 年レイヤーを持つ。
- 近接月のみ遅延読み込み。
- 起動時は当月に着地。

### 3.3 その日ビュー

- ボードの写真をタップすると、その日を開く。
- おくった自分の猫と、とどいた猫を日付で束ね、送った→届いたの時系列で見せる。
- 融合した1枚にはしない。
- 再オープンはタップした写真からの morph。
- フルの蝋セレモニーは20:00到着時のみ。
- とっておくはフルスクリーン単写真ビューに置く。

### 3.4 貼るモーション

- 写真がボードに降りて落ち着く。
- トリガはカメラと到着。
- 着地は共通。
- ゲーム化しない。

### 3.5 空状態

- コピーは「ねがおが、すこしずつ ふえる」。
- セルも数字も出さない。

### 3.6 コピー

- 空っぽ: ねがおが、すこしずつ ふえる
- 貼るボタン: ねがおを おくる
- 到着側: よその ねがお が とどく
- 各写真: 日付＋おくった / とどいた
- 一日の仕切り: よる
- 「ねこだより」は名詞・名前として使う。

## 4. うちのこ

- 自分の猫だけを時間で眺める深いアルバム。
- まいにちとは役割分担する。
- コレクション/アルバムへのタブ分割はしない。

## 5. 横断ルール

- ストリーク、各種カウント、フィード、ランキングを作らない。
- コピーはわかりやすく、眠りの温かさを持つ。
- 「手紙」を使わない。
- 「一生」「有限」を言語化しない。
- 損失回避・人質的言い回しをしない。
- フォントは Klee One と Zen Kaku Gothic New。
- トークンと paper-and-ink canon に従う。
