# ねてるねこ UI改修 spec v1.1（デザイン整理編）

## 0. 確定コピー

- 配達なし/片側のみの日のカード: **「おたよりは ありませんでした」**
- state1 封筒の時刻表記: **「よる8じに とどきます」**

## 1. 作業順序

1. タブバー
2. ホーム state1
3. アルバム「まいにち」= ひらいた手紙カード
4. ねこページ
5. 設定の薄文字 + 復元ボタン

各タスクは既存の構造・命名・スタイルトークンに合わせる。新規の色は `ux-performance-visual-v1.0` のコントラスト方針に従い、グレー直書きを増やさない。

## 2. タブバー

- 全画面で1番目タブを「きょう」に統一する。
- タブ定義を単一ソースに集約し、画面ごとの別ラベルを撤去する。
- 選択中タブのみラベルを表示し、非選択タブはアイコンのみ。
- タブ構成は `きょう` / `アルバム` / `ねこ`。
- 各スクロールビュー下端に `タブバー高さ + safe-area-inset-bottom` 分の padding を確保する。

## 3. ホーム state1

- 封筒は不透明で明確な閉じた封筒として描画する。
- 封筒の下またはラベルに「よる8じに とどきます」を表示する。
- カメラ枠はタップ可能に見える白系の塗り + 影、または枠内塗りボタンにする。
- 既存の「とる」ラベルは維持する。
- state1 以外のレイアウトと状態機械は変更しない。

## 4. アルバム「まいにち」

- 点線を廃止する。
- 「ほかのねがお」横長帯 + 丸サムネを廃止する。
- 1日 = 1枚の「ひらいた手紙」カードに再構成する。
- カード内は左=むぎ、右=どこかのこ。各写真の下にラベルを置く。
- どこかのこが届かなかった日は右側に「おたよりは ありませんでした」を表示する。
- うちのこタブは変更しない。

## 5. ねこページ

- 「家族になって ◯日」を主役数字として大きく表示する。
- 誕生日が入力済みなら「たんじょうびまで あと◯日」を表示する。
- 誕生日当日は「きょうは ◯◯の たんじょうび」を表示する。
- 「たんじょうびには とくべつな おたよりが とどきます」は予告文に留める。
- 柄タグの近傍または押下時に「ずかんで つかわれます」を表示する。

## 6. 設定

- 設定画面全体の薄いグレー文字を本文/補助トークンへ置換する。
- 「アカウントからこの端末に復元」ボタンを、活性/非活性が明確な見た目にする。
- 置換箇所を報告する。

## 7. 検証

各タスク後に `npm run typecheck` / `npm run build` / 関連E2Eを通す。最終的に `npm run e2e` を通す。

## 8. スコープ外

- うちのこタブのレイアウト
- home state2〜4 と状態機械
- モデレーション関連
- `ux-performance-visual-v1.0` の残タスク
