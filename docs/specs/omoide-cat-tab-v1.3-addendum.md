# ねてるねこ 実装仕様: 思い出が届く & ねこタブ改訂

版: v1.3 追補  
日付: 2026-06-13  
対象: Codex 実装

前提: 夜8時(JST)の単一配達、手紙/封筒のメタファ、静けさ。ポイント、ミッション、ログインボーナス、ストリークは禁止。

## 0. 範囲

この追補では次の4ブロックを追加・改訂する。

1. 思い出が届く: 過去のねがおが自分への一通として夜の配達で届き、ひらくと全画面で現像される。
2. 配信ポリシー: 頻度、トリガー、遡り距離、初期仕込み、喪失時の安全装置。
3. ねこタブ改訂: 届いた思い出が住む場所と関係の記録にする。アルバムとは分離する。
4. コピー、計測イベント、廃止リスト。

設計の核: ねこタブの主役は写真ではなく、猫という存在と関係の時間。「うちの子の写真を日付順に並べたもの」に見えたらアルバムへの侵食として扱う。

## 1. 思い出が届く

過去に保存したねがおから1枚を選び、「1年前のきょう」「1ヶ月前のきょう」「1週間前のきょう」などの文脈を添えて、自分の過去からの一通として届ける。

配信チャネル:

- 夜8時(JST)の配達で、日々の交換便とは別便として届く。
- 交換便を置き換えない。
- ホームに思い出の封筒が到着し、ひらくと全画面の現像演出に入る。
- 以後はねこタブの文箱に残る。

状態:

- 到着: ホーム。封蝋「思」と消印つきの封筒。
- ひらく: 全画面モーダル。写真がぼやけからくっきり現像される。
- 保管: ねこタブの文箱。何度でも開き直せる。

## 2. 配信ポリシー

- 毎日出さない。
- 見せる価値のある過去のねがおがある日にだけ生成する。
- 対象日に保存済みのねがおが1枚以上あること。
- 候補がなければ配信しない。
- 遡り距離は、1週間前、1ヶ月前、半年前、1年前、N年前の順で伸びる。
- 目標頻度は月1から3通。
- 上限はおよそ7日に1通。直近7日以内に思い出を配信済みなら出さない。
- 加入後7から10日目に最初の1通を配信する。内容は「はじめての ねがお」または「猫と 1週間」。

候補優先順位:

1. 過去の同じ日。
2. 節目。
3. 季節の谺。

喪失の安全装置:

- 飼い主が「おわかれ」を記したら、お祝い調の自動配信を停止または静かなトーンにする。
- 「思い出を しばらく お休みする」を用意する。
- 「この日を 見せない」を用意する。
- 「思い出を 受け取らない」を用意する。
- 強制ポップアップは禁止。開かずに「そっと しまう」で断れる。

## 3. 画面・フロー

### ホーム

- 思い出がある夜のみ、ホームに到着した封筒を表示する。
- 封筒は封蝋「思」と消印で交換便と区別する。
- タップで全画面モーダルへ遷移する。
- 交換便を開封済みの夜は補足行を出してよい。

### 思い出の全画面

構成:

- アイ: 思い出が、とどきました
- 日付: 「N年前の、きょう。」と実日付
- 現像される一枚
- 声
- ブリッジ
- 問いかけ
- CTA
- そっと しまう
- 守る囁き一行

現像演出:

- blur(14px) から 0。
- opacity 0 から 1。
- 微 scale。
- 約1.5秒。
- prefers-reduced-motion では即時表示。

声の生成ルール:

- システムが実際に持つ事実のみ使う。
- 日付、経過日数、季節、ねぞう分類済みならその語のみ。
- 「窓辺で」など未確認の具体は捏造しない。

### ねこタブ

構成:

1. ヘッダー: 代表写真、名前、「猫と 暮らして N 日」、カウンター、共有導線。
2. とどいた思い出: 文箱。手紙が横スクロールで積もる。窓から寝顔を覗かせる。
3. 猫との日々: 最近は開き、過去は年で畳む一本の糸。
4. 守るの安堵: フッター一行。

廃止:

- 「原本で おあずかり中」バッジ。
- 「一生」表記。
- 旧ねこタブの平坦な無限スクロール。

## 4. 確定コピー

ホーム:

- 今夜は、思い出が とどきました
- 過去から、一通。
- 1年前の むぎから 届きました。
- はじめての ねがお が 届きました。
- そっと ひらく
- ひらくと、その日の ねがおが 現像されます
- 今夜の交換便は、もう ひらきました

全画面:

- 思い出が、とどきました
- N年前の、きょう。
- YYYY年M月D日 ・ Xよう日
- あれから、N日。
- あれから、N年。
- きょうの むぎは、どんな ねがお？
- いまの むぎを のこす
- きょうの むぎを みる
- そっと しまう
- この一枚は、元の画質のまま とどいています。

ねこタブ:

- むぎと 暮らして N 日
- N回目の季節
- N とどいた思い出（通）
- N ねがお（枚）
- とどいた思い出
- 配達で とどくたび、この箱に 積もっていきます
- むぎとの 日々
- 今月の むぎ
- これまで
- はじめての思い出は、これから届きます。
- ここから、むぎとの 日々が はじまります。

安全装置:

- 思い出を しばらく お休みする
- この日を 見せない
- 思い出を 受け取らない
